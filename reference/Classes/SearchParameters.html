SearchParameters

@objc public class SearchParameters: Query

A high-level representation of a query’s filter.

This class allows manipulating facet and numeric filters individually, then combining them into a string suitable for use in Query.filters.

Note

Tags are not handled. Please use facets instead, as they are more powerful.
